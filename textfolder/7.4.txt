デバックフォルダを作っている
デバックはあちこちで書くからインクルードパスを指定してあげる

命令群
start		デバック用のDrawScreenのclearを行っているだけ（GameSceneとほぼ同じ）
Runが走るwhileの最初に呼んでいる

add		SceneMngのキューへの追加を行う関数
SceneMngのDrawの最初に呼んでいる
	
setUp		デバック用のスクリーン作成関数	引数　int alpha
スクリーンのサイズはゲームのスクリーンではなく画面のスクリーンを作成する
この関数の最後にセットアルファも呼ぶ

setAlpha	透明度の設定用関数	引数　int alpha

初期化のデバックスクリーンは-1を指定

セットアップ内にはディーエックスリブ用の命令があるから
今の構造でコンストラクタに書くと必ずコケるからSceneMngのSysInit内で
呼べばdxlib命令が使えるから大丈夫
#includeと生成の順番大事


ドロー系はスクリーンのセットを行って、アルファチャンネル設定後、各ドローのやつを呼んで
アルファチャンネルを戻してスクリーンを書き込む
この中のドローではゲームスクリーンのオフセットも渡してあげないと本来の場所とずれる
アルファー　alpha
ノーブレンド　0
