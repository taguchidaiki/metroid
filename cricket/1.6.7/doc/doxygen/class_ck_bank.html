Loads a bank asynchronously.

This will return NULL if the bank cannot be loaded. When loading a bank asynchronously, you should also check isFailed() after loading to make sure no errors occurred.

If the bank file is embedded in a larger file, specify the offset and length of the embedded data; otherwise, leave them both as 0.
