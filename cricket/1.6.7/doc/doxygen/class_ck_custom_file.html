Detailed Description

Custom file base class.

If you want to override the low-level details of how files are read (for example, to read files over a network, or to decrypt encrypted files as they are read), create a subclass and register a handler with CkSetCustomFileHandler().

This will work for bank files, and for .cks and Ogg Vorbis streams, but not for other stream types such as MP3, AAC, etc.
