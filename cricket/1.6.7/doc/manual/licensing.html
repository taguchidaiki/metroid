Licensing

Cricket Audio is available under two licenses: the free license, and the (non-free) source code license.

Free license

This free license allows you to use the binary form of Cricket Audio for no cost, with a few requirements:
  • Include the following message somewhere in your app, visible to users (for example, on a credits screen):

    Built with Cricket Audio
    www.crickettechnology.com

    You may optionally include our logo (in .svg or .png format).

  • Do not distribute our software to anyone, or post it for download on any site; refer them to our website instead.

  • This license does not permit you to distribute Cricket Audio in an SDK, library, or other software development product. Contact us at for additional licensing options.

  • If you release an app that uses Cricket Audio, you must let us know; we will include a link to it on our customers page.

Source code license

The source code license allows you to use the binary form of Cricket Audio and provides access to the Cricket Audio source code.  Unlike the free license, there is no requirement for a credit in the final product (though you can include one if you want), and you do not need to notify us of your releases (though you're free to do that too).

The source code license also has a few requirements:
  • Do not distribute our software to anyone, or post it for download on any site; refer them to our website instead.

  • Do not distribute the source code to anyone who does not have an active source code license.

  • This license does not permit you to distribute Cricket Audio in an SDK, library, or other software development product. Contact us at for additional licensing options.

Source code licenses can be purchased on our website.
