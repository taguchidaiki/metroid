Building Cricket Audio from source

Users who have purchased a source license can build Cricket Audio from source.

iOS

The iOS version is built with Xcode on OS X. The Xcode workspace is in src/ck.xcworkspace.

Android

The Android version can be built on OS X, Windows, or Linux using GNU make. On Windows, you will need to install Cygwin.

You will need to download and install the Android NDK and Android SDK, and set the environment variables CK_ANDROID_NDK_DIR and CK_ANDROID_SDK_DIR to those directories.

To build, first change to the ck/src/ck directory, then run

    ./gradlew build

OS X

The OS X version is built with Xcode on OS X. The Xcode workspace is in src/ck.xcworkspace.

Windows

The Windows version is built with Visual Studio on Windows 10. The solution files are in src/ck; choose the version that corresponds to the version of Visual Studio you are using.  For example, use ck_win_vs2017.sln for Visual Studio 2017.

Linux

The Linux version can be built using GNU make.

To build, first change to the ck/src/ck directory, then run

    make CK_TARGET_OS=linux

which will build the debug version for all available architectures.

To build the release version, run

    make CK_TARGET_OS=linux CK_CONFIG=release

To build for only one architecture, run

    make CK_TARGET_OS=linux CK_ARCH=arch

where arch is one of the supported architectures (x86 or x64).
